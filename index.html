Quick Start

# Clone repository
git clone https://github.com/constella-team/constella.git
cd constella

# Configure environment
cp .env.example .env
# Edit .env with your Plex credentials

# Start with Docker
docker compose up -d

# Access web UI
open http://localhost:8000
